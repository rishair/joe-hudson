# Full podcast episode audio download

## Hypothesis

Downloading all 167 podcast episodes from the iTunes API URLs via curl will produce a complete set of MP3 files.

## Method

Used the download script from the method section, reading `meta/raw/itunes-episode-list.tsv` and downloading each episode to `transcripts/{date}_{title}/{title}.mp3` with 1-second sleep between downloads. Files already present were skipped.

## Result

**167 of 167 episodes downloaded.** 136 new downloads + 31 skipped (from earlier test runs). All 136 new downloads returned HTTP 200. No failures.

Episodes span October 2020 to May 2026. The final episode downloaded was "The Fear of Being Seen: Overcoming Shame, Invisibility, and Social Anxiety" (2026-05-08).

## Analysis

The podcast download pipeline works at full scale. All Simplecast CDN URLs from the iTunes API resolved successfully, including episodes from 2020. No authentication, rate limiting, or expired URLs were encountered.

**Boundary conditions**: Tested once on 2026-05-20. URLs contain tracking parameters but no expiration tokens. If Simplecast changes CDN structure or the podcast moves hosts, URLs may break. The iTunes API catalog (`meta/raw/itunes-episode-list.tsv`) is the source of truth for episode URLs.
