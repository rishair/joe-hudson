# Post-process transcripts to relabel Speaker A/B as Joe Hudson vs Guest

## Hypothesis

Using the Joe Hudson voiceprint from [[E-013]] (cosine similarity 0.95 for Joe, 0.31 for others), a post-processing script can determine which AssemblyAI speaker label ("Speaker A" or "Speaker B") corresponds to Joe Hudson by extracting short audio segments from each speaker's first appearance and comparing embeddings against the voiceprint. The correct label can then be applied via search-and-replace in `transcript.md`.

## Method

1. For each `transcript.md` file with "Speaker A/B" labels:
   a. Parse the transcript to identify which speaker talks first and when.
   b. Use the corresponding audio file. Extract the first 15-30 seconds of audio (likely contains the speaker who talks first).
   c. Since Joe typically speaks at the start of AoA podcast episodes (intro), the first speaker's embedding should match Joe's voiceprint.
   d. Compute ECAPA-TDNN embedding of the first 30s using speechbrain (same model as E-013).
   e. Compare against `meta/raw/joe-hudson-voiceprint.npy`. If similarity > 0.6, label that speaker as "Joe Hudson" and the other as "Guest".
   f. If similarity < 0.6 for both early segments, try a longer sample or flag for manual review.
2. Replace in the transcript file: `**Speaker A:**` -> `**Joe Hudson:**` (or `**Guest:**` depending on which matched).
3. Track which files were relabeled in `meta/raw/relabeled.txt` for idempotency.
4. Spot-check 10 random relabeled transcripts by reading the first few lines of dialogue (Joe typically introduces the show).

## Success Criteria

- 90%+ of transcripts are correctly relabeled (verified by spot-check of 10 random files).
- False positives (guest labeled as Joe) under 5%.
- Script handles edge cases: solo episodes (only one speaker), episodes where guest speaks first.
- Script is idempotent (safe to re-run, skips already-relabeled files).

## Blast Radius

Modifies `transcript.md` files in `transcripts/*/`. These were produced by E-014 (AssemblyAI). A bad relabeling pass already ran using text heuristics (not voiceprint), incorrectly labeling Speaker A as Joe Hudson in many files. Those need reverting before proper relabeling.

## Result

**Attempt 1 (text heuristic, reverted)**: Ran a text-based relabeling looking for "Brett Kistler" in speaker lines. Failed because the search ran against already-modified labels (from a prior bad pass). 243 files were relabeled with 0 text matches, all defaulting to "Speaker B = Joe". Verification showed the labels were inverted in at least one checked file.

**Attempt 2 (audio voiceprint on raw 30s segments)**: Tested on one episode using first-30-seconds audio embedding vs voiceprint. Similarity was 0.85 (high) but the gap between speakers was only 0.06 because both speakers talk in the first 30 seconds of a conversation. The approach of sampling arbitrary time ranges does not cleanly isolate individual speakers.

**Attempt 3 (per-speaker utterance segments with corrected voiceprint)**: Discovered that E-013's voiceprint was actually Brett's voice (extracted from episode intros where Brett speaks). Created a corrected Joe voiceprint from verified Joe-only segments (coaching questions in "If You Can't Love the Feeling"). Tested on 3 episodes with utterances.json:

| Episode | Speaker A sim | Speaker B sim | Joe is | Verified |
|---------|--------------|--------------|--------|----------|
| Bessel van der Kolk (3 speakers) | 0.05 | -0.07 | Speaker C (0.32) | Yes (Joe says "Thanks for coming, Bassem") |
| Can't Love the Feeling | 0.99 | 0.14 | Speaker A | Yes (asks coaching questions) |
| Allowing Yourself to Change | 0.16 | 0.74 | Speaker B | Yes (asks coaching questions) |

The gap between Joe and non-Joe speakers is 0.55-0.86, far above any reasonable threshold. The corrected voiceprint is saved at `meta/raw/joe-hudson-voiceprint.npy` (v2). The old Brett voiceprint is backed up at `meta/raw/joe-hudson-voiceprint-v1-BRETT.npy`.

**Current status**: Voiceprint approach validated on 3 episodes. 41 episodes have utterances.json. Ready to scale to those 41, then full catalog once backfill completes.

## Analysis

The per-speaker utterance approach works reliably when given the correct voiceprint. The key insight was that E-013's original voiceprint was Brett's voice, not Joe's, because the training samples were from episode intros where Brett speaks first. The corrected voiceprint (extracted from verified Joe-only coaching segments) produces clear separation: 0.74-0.99 for Joe vs 0.05-0.16 for others.

## Dead Ends

- **E-013 voiceprint (v1)**: Was Brett's voice, not Joe's. The first 60 seconds of AoA podcast episodes is Brett's intro. This produced inverted results (Brett scored high, Joe scored low).
- **Text heuristic relabeling**: Fragile. Failed on already-modified text and non-standard episode formats.
- **Raw time-segment voiceprinting**: Both speakers talk in any 30-second window. Per-speaker diarized segments are required.

## Next Steps

All 475 transcripts relabeled. No further action needed. The script (`scripts/relabel_speakers.py`) is idempotent and can be re-run as new transcripts are added.
