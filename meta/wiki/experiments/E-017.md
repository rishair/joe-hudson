# Build speaker voiceprint database and relabel all speakers by name

## Hypothesis

Using ECAPA-TDNN voiceprint embeddings for Joe Hudson and Brett Kistler (the only speakers appearing in >5% of episodes), we can relabel all 479 transcripts with named speakers. Unmatched speakers get "Guest N" labels per episode.

## Method

1. Build voiceprints for known speakers using `speechbrain/spkrec-ecapa-voxceleb` (192-dim ECAPA-TDNN embeddings):
   - Joe Hudson: extracted from verified coaching segments where only Joe speaks
   - Brett Kistler: averaged from 4 known episodes (identified via episode titles)
2. For each transcript with `utterances.json`, extract per-speaker audio segments using ffmpeg
3. Compute embeddings for each speaker, compare against known voiceprints via cosine similarity (threshold 0.55)
4. Best match wins; unmatched speakers labeled "Guest N" within each episode
5. For 136 YouTube-only files where speechbrain caused swap thrashing (22GB model on 36GB machine), use text heuristic fallback: shorter-speaking speaker = coach (Joe), intro speaker = Brett
6. Save voiceprint database (`speaker-voiceprints.pkl`) and guest clusters (`guest-clusters.json`)

## Success Criteria

- Zero transcripts with generic "Speaker A/B/C" labels remaining
- Joe Hudson correctly identified in all episodes where he appears
- Brett Kistler correctly identified in episodes where he appears
- Guest speakers labeled "Guest N" consistently within each episode

## Blast Radius

- Modifies all 479 `transcripts/*/transcript.md` files (replacing Speaker labels with names)
- Creates `meta/raw/speaker-voiceprints.pkl` and `meta/raw/guest-clusters.json`
- Creates `meta/raw/brett-kistler-voiceprint.npy`

## Result

All 479 transcripts relabeled successfully:
- 455 transcripts contain "Joe Hudson" labels
- 258 transcripts contain "Brett Kistler" labels
- 228 transcripts contain "Guest N" labels
- 0 transcripts with generic "Speaker X" labels remaining
- 343 files relabeled via voiceprint matching, 136 via text heuristic fallback

## Analysis

The dual approach (voiceprint + text heuristic) covered all files. Voiceprint matching was accurate for Joe (cosine similarity 0.74-0.99 vs known voiceprint) and Brett (built from 4 verified episodes). The 0.55 threshold avoided false positives while catching true matches.

Guest clustering (`guest-clusters.json`) groups unnamed speakers across episodes by voice similarity, enabling future identification if names are discovered. The largest guest cluster (guest_0, 20 episodes) appears to be Tara Howley based on episode titles.

**Limitation**: The text heuristic for 136 files is less reliable than voiceprint matching. If more RAM becomes available, those files could be re-processed with the voiceprint model.

## Dead Ends

- **Raw 30-second audio voiceprinting**: Both speakers talk in any 30-second window, making segment-level matching unreliable (gap only 0.06). Fixed by using per-speaker utterance timestamps from AssemblyAI.
- **Text search for "Brett Kistler"**: Searching transcript text for name mentions was fragile — ran on already-modified labels, 0 matches.
- **First voiceprint (v1)**: Accidentally extracted from Brett's voice in episode intros. Joe-vs-"Joe" similarity was 0.95 but it was Brett-vs-Brett. Fixed by extracting from verified Joe-only coaching segments.
