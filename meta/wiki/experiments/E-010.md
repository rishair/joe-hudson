# Full YouTube channel audio download

## Hypothesis

Running yt-dlp against the full AoA YouTube channel with sleep intervals will download all 329 videos as MP3 audio without rate limiting or 403 errors, based on the successful 5-video test in [[E-008]].

## Method

1. Run the full channel download as a background process:
   ```
   yt-dlp -x --audio-format mp3 --embed-metadata \
     --output "transcripts/%(upload_date>%Y-%m-%d|)s_%(title)s/%(title)s.%(ext)s" \
     --download-archive archive.txt \
     --sleep-interval 3 --max-sleep-interval 10 \
     --write-info-json \
     "https://www.youtube.com/@ArtofAccomplishment/videos" \
     2>&1 | tee meta/raw/youtube-download.log
   ```
2. The `--download-archive archive.txt` flag skips the 5 already-downloaded test videos.
3. The `--write-info-json` flag saves per-video metadata alongside the MP3.
4. Monitor `archive.txt` line count periodically to track progress.
5. After completion, count total MP3 files in `transcripts/` and compare against the 329 expected from [[E-001]].

## Success Criteria

- 320+ MP3 files downloaded (allowing for a few that may fail due to age-restricted or removed videos).
- No sustained rate limiting (occasional retries are acceptable).
- All files are valid MP3s (spot-check 5 random files with `file` command).
- Total download size is in the expected range (~5-6 GB).
