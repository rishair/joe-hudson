# AssemblyAI comparison baseline

## Hypothesis

AssemblyAI with speaker diarization will produce noticeably better transcripts and diarization than WhisperX on the same test episodes, justifying the API cost for the full catalog.

## Method

1. Only run this experiment if [[E-005]] results show quality issues (WER > 10% or DER > 20%).
2. Take the same 2-3 test episodes used in [[E-005]].
3. Upload each to AssemblyAI with `speaker_labels: true` using their Python SDK.
4. Compare output side-by-side with WhisperX output on the same spot-checked segments.
5. Record: WER difference, diarization accuracy difference, and total API cost.

## Prerequisites

- [[E-005]] must complete first. This experiment is conditional on local quality being insufficient.
- Requires an AssemblyAI API key (free tier provides 100 hours).

## Success Criteria

- AssemblyAI produces measurably better output (WER or DER improvement of 5+ percentage points) on the same test segments.
- If AssemblyAI is not meaningfully better, the local approach wins and this experiment confirms it.

## Cost Estimate

3 episodes at ~60 min each = 3 hours. At $0.17/hr with diarization = ~$0.51. Well within budget for a comparison test.
