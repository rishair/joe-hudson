# Full podcast episode archive via PodcastIndex API

## Hypothesis

The PodcastIndex.org API has indexed the full history of the Art of Accomplishment podcast (~160 episodes) and can provide direct audio URLs for bulk download, bypassing the Simplecast RSS feed's ~25-episode window.

## Method

1. Register for a free PodcastIndex.org API key at https://api.podcastindex.org.
2. Search for the Art of Accomplishment podcast by feed URL (`https://feeds.simplecast.com/zksImfUP`) using the `/podcasts/byfeedurl` endpoint.
3. Use `/episodes/byfeedid` with the resulting feed ID to retrieve all episodes. The API returns results in pages; iterate until all episodes are retrieved.
4. Extract audio enclosure URLs, episode titles, and publish dates from the API response.
5. Download all audio files using curl or wget, organized into `transcripts/{YYYY-MM-DD}_{title}/` directories.
6. Create a `MEDIA.md` stub in each directory with title and date.
7. Verify: count downloaded episodes vs expected ~160. Spot-check 3-5 files for valid audio.

**Fallback**: If PodcastIndex does not have the full history, try scraping `https://www.artofaccomplishment.com/podcast` for episode links.

## Success Criteria

- PodcastIndex API returns 150+ episodes for this feed.
- Audio URLs resolve and download successfully.
- Downloaded MP3s are valid and playable.
- All episodes are stored in the target directory structure with MEDIA.md stubs.
- Failure condition: PodcastIndex has fewer episodes than the RSS feed, or audio URLs are expired/gated.
