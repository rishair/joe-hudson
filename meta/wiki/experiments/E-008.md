# yt-dlp audio download from AoA YouTube channel

## Hypothesis

yt-dlp with exported browser cookies and sleep intervals can reliably download audio-only files from the Art of Accomplishment YouTube channel without triggering 403 errors or rate limiting.

## Method

1. Install yt-dlp: `brew install yt-dlp` or `pip install yt-dlp`. Verify with `yt-dlp --version`. Install FFmpeg if not already present (`brew install ffmpeg`).
2. Export YouTube cookies from a logged-in browser session using the "Get cookies.txt locally" extension. Save as `cookies.txt`.
3. Test single video download (audio-only):
   ```
   yt-dlp -x --audio-format mp3 --embed-metadata \
     --cookies cookies.txt \
     "https://www.youtube.com/@ArtofAccomplishment/videos" \
     --playlist-items 1
   ```
4. If single video works, download 3-5 videos with sleep intervals:
   ```
   yt-dlp -x --audio-format mp3 --embed-metadata \
     --output "transcripts/%(upload_date)s_%(title)s/%(title)s.%(ext)s" \
     --download-archive archive.txt \
     --cookies cookies.txt \
     --sleep-interval 5 --max-sleep-interval 30 \
     "https://www.youtube.com/@ArtofAccomplishment/videos" \
     --playlist-items 1-5
   ```
5. Verify: check that MP3 files are valid, have embedded metadata, and are stored in the correct directory structure.

## Success Criteria

- yt-dlp installs and runs without errors.
- At least 3 of 5 test videos download successfully as MP3 audio files.
- No 403 errors or rate-limit blocks during the test.
- Output files are valid MP3s with embedded metadata (title, date).
- Failure condition: persistent 403 errors even with cookies, or audio quality is unusable.
