# Scrape AoA website podcast page for complete episode archive

## Hypothesis

The Art of Accomplishment website or an alternative public API hosts a browsable archive of all podcast episodes, including those no longer in the RSS feed window.

## Method

Four sources were tested:

1. **AoA website** (https://www.artofaccomplishment.com/podcast): Fetched with WebFetch.
2. **Simplecast RSS feed** (https://feeds.simplecast.com/zksImfUP): Fetched directly.
3. **Podnews** (https://podnews.net/podcast/iornn/episodes): Fetched with WebFetch.
4. **iTunes Lookup API** (`https://itunes.apple.com/lookup?id=1540650504&media=podcast&entity=podcastEpisode&limit=200`): Fetched with curl and parsed with Python.

## Result

| Source | Episodes found | Audio URLs? | Notes |
|--------|---------------|-------------|-------|
| AoA website | 12 | No (JS "Load More") | Webflow site, renders episodes via JavaScript. Only 12 visible without JS execution. |
| RSS feed | ~20 | Yes | Simplecast feed is windowed. Only recent episodes. |
| Podnews | 0 (JS-rendered) | No | Page loads episodes dynamically. Static HTML contains only loading placeholders. |
| **iTunes API** | **167** | **Yes, all 167** | Complete catalog from Oct 2020 to May 2026. Every entry has a direct Simplecast audio URL. |

The iTunes Lookup API returned 167 episodes with title, date, duration, and direct audio download URL for every episode. Apple Podcasts confirms 167 episodes total, matching the API count exactly.

**Episode distribution by year**: 2020: 6, 2021: 31, 2022: 31, 2023: 27, 2024: 30, 2025: 28, 2026: 14.

The oldest episode is "Introduction to VIEW -- Connection Course Series #1" (2020-10-26). The newest is "The Fear of Being Seen" (2026-05-08).

Data saved to `meta/raw/itunes-episode-list.tsv` (167 rows with date, title, duration_ms, episode_url).

## Analysis

The iTunes Lookup API is the winning approach. It is public, requires no authentication, and returns the complete catalog with direct audio URLs. The Simplecast audio URLs in the response are the same CDN URLs that the RSS feed provides, so they are direct MP3 downloads.

The PodcastIndex API (identified in [[R-003]]) would also work but requires registration and authenticated requests. The iTunes API is simpler and already proven.

The AoA website and Podnews both rely on JavaScript rendering, making them unsuitable for static scraping. The RSS feed is deliberately windowed by Simplecast configuration.

**Boundary conditions**: The iTunes API limit parameter caps at 200. Since there are 167 episodes, this is not a problem. If the podcast exceeds 200 episodes in the future, pagination or an alternative source will be needed. Audio URLs contain tracking parameters and may expire; testing download of an older episode URL is needed to confirm they remain valid.

## Dead Ends

- **AoA website**: Webflow-based, JavaScript "Load More" pagination. Cannot be scraped without a headless browser.
- **Podnews**: Entirely JS-rendered. The static HTML is just loading placeholders.
- **RSS feed**: Only ~20 episodes due to Simplecast's feed limit setting. This is the podcast owner's configuration, not a technical limitation.
