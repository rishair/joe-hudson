# Scrape AoA website podcast page for complete episode archive

## Hypothesis

The Art of Accomplishment website at https://www.artofaccomplishment.com/podcast hosts a browsable archive of all podcast episodes, including those no longer in the RSS feed window. Fetching this page (and any pagination) will yield a complete episode list with titles, dates, and per-episode URLs. The website is built on a CMS (likely Squarespace or similar) that renders episode listings server-side or via a paginated API.

## Method

1. Fetch https://www.artofaccomplishment.com/podcast using WebFetch. Extract: episode titles, publication dates, episode URLs, and any episode numbers visible.
2. Check for pagination: look for "load more", "next page", or numbered page links. If paginated, follow all pages.
3. If the page uses JavaScript-rendered content that WebFetch cannot capture, try an alternative: search for an underlying API endpoint (e.g., Squarespace collection JSON, Simplecast episode API) in the page source.
4. Alternative fallback: use the Podnews listing at https://podnews.net/podcast/iornn/episodes which may have a more complete archive.
5. Cross-reference the count against the RSS feed's episode 160 numbering and the Apple Podcasts listing to assess completeness.
6. Save raw results to `meta/raw/website-podcast-listing.json` or `.tsv`.

## Success Criteria

- **Success**: A list of 150+ episodes (matching the known ~160 numbered episodes plus unnumbered Bite-Sized Teaching shorts) with title and date for each. URLs are a bonus.
- **Partial success**: A list of 100+ episodes but with gaps in early episodes.
- **Failure**: The website only shows the same ~25 episodes as the RSS feed, or the page cannot be scraped.
