# Use yt-dlp to extract full video list from the AoA YouTube channel

## Hypothesis

Running `yt-dlp --flat-playlist` against the AoA YouTube channel URL will produce a complete list of all public videos with metadata (title, upload date, duration, video ID, URL). This should capture all YouTube content in a single command without downloading any media files.

## Method

1. Installed yt-dlp via `brew install yt-dlp` (version 2026.03.17).
2. Ran flat-playlist extraction: `yt-dlp --flat-playlist --print "%(id)s\t%(title)s\t%(upload_date)s\t%(duration)s\t%(url)s" "https://www.youtube.com/@ArtofAccomplishment/videos"`
3. Checked playlists: `yt-dlp --flat-playlist --print "%(playlist_title)s\t%(id)s\t%(title)s" "https://www.youtube.com/@ArtofAccomplishment/playlists"`
4. Tested full JSON mode (`yt-dlp -j --no-download`) on a single video to check date availability.

## Result

**329 videos** extracted to `meta/raw/youtube-channel-listing.tsv`. No errors.

Fields obtained per video: video ID, title, duration (in seconds), and full YouTube URL. Upload dates returned as "NA" in flat-playlist mode for all 329 videos. This is a known yt-dlp limitation: flat-playlist mode skips per-video API calls, so date fields are unavailable.

Full JSON mode (`-j --no-download`) on a single test video confirmed that upload dates are available when fetching full metadata. Running this for all 329 videos would require 329 individual API calls and would take considerable time (estimated 15-30 minutes).

**21 playlists** found, saved to `meta/raw/youtube-playlists.tsv`. Playlist names include:

Playlists are organizational groupings of the same 329 videos, not additional content.

## Analysis

The experiment succeeded at its primary goal: a complete listing of all public videos on the channel with IDs, titles, durations, and URLs. The 329-video count is plausible for a channel active since 2020.

Upload dates are missing from the flat-playlist output. Two options to obtain them:
1. Run full JSON metadata extraction for all 329 videos (slow but complete).
2. Use the YouTube Data API v3 with a batch request (faster, requires API key).

The 21 playlists provide useful categorization metadata. The "Interviews with Joe" playlist is worth cross-referencing with [[E-003]] (guest appearances catalog) to distinguish between AoA-hosted interviews and external guest appearances.

Duration data is present and in seconds. Videos range from short clips (~400s / 7 min) to long-form content (~3400s / 57 min).

## Dead Ends

Flat-playlist mode cannot retrieve upload dates. This is inherent to how yt-dlp works in this mode. It is not an error or misconfiguration.
