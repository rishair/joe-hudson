# Use yt-dlp to extract full video list from the AoA YouTube channel

## Hypothesis

Running `yt-dlp --flat-playlist` against the AoA YouTube channel URL will produce a complete list of all public videos with metadata (title, upload date, duration, video ID, URL). This should capture all YouTube content in a single command without downloading any media files.

## Method

1. Verify yt-dlp is installed: `yt-dlp --version`. If not installed, install via `brew install yt-dlp` or `pip install yt-dlp`.
2. Run: `yt-dlp --flat-playlist --print "%(id)s\t%(title)s\t%(upload_date)s\t%(duration)s\t%(url)s" "https://www.youtube.com/@ArtofAccomplishment/videos" > meta/raw/youtube-channel-listing.tsv`
3. Count lines in output to get total video count.
4. Inspect output for: (a) presence of upload dates, (b) presence of durations, (c) any missing fields. Not all fields may be available in flat-playlist mode.
5. If flat-playlist mode lacks dates/durations, try: `yt-dlp -j --no-download "https://www.youtube.com/@ArtofAccomplishment/videos"` for full JSON metadata per video. This is slower but richer.
6. Also check playlists: `yt-dlp --flat-playlist --print "%(playlist_title)s\t%(id)s\t%(title)s" "https://www.youtube.com/@ArtofAccomplishment/playlists"` to see if content is organized into playlists (e.g., the Masterclass series).

## Success Criteria

- **Success**: A complete TSV or JSON file listing all public videos on the channel, with at minimum: video ID, title, and URL for each. Dates and durations are a bonus. Count should be verifiable against what the channel page reports.
- **Failure**: yt-dlp cannot access the channel, returns zero results, or returns a clearly incomplete list (e.g., only the most recent page of results).
