# How to bulk-download YouTube channels and podcast feeds in 2026

## Question

What tools and methods currently work for bulk-downloading all videos from a YouTube channel and all episodes from a podcast RSS feed? Have yt-dlp or similar tools been affected by recent YouTube changes?

## Why This Matters

This is the primary execution method for [[G-003]]. If yt-dlp is broken or rate-limited, we need alternatives. If podcast RSS feeds are straightforward, we can start there while solving YouTube issues. The answer determines which experiments to run.

## Constraints and Scope

- In scope: yt-dlp (current version and known issues), youtube-dl forks, podcast feed downloaders, any other bulk download tools.
- Check: does yt-dlp still work with YouTube as of mid-2026? Are there rate limits or blocks? What format options are available (audio-only vs video)?
- For podcasts: can we download directly from RSS feed URLs? Do Apple Podcasts / Spotify have direct download options?
- Out of scope: building custom scrapers from scratch (try existing tools first).
- The agent should check yt-dlp's GitHub issues/releases for recent changes, and test if `yt-dlp --version` is installed locally.
