# What's the scary thing that's true for me right now?

## The Question

- "What's the scary thing that's true for me right now?"
- "Where are your fear and your truth together right now?"

## When Joe Uses It

This is Joe's compass question for vulnerability. It can be used at any moment, in any conversation, to find the vulnerable move. It works both as a coaching question (asked of someone else) and as a self-inquiry practice.

Joe uses it when someone is stuck, performing, or hiding behind a persona. It cuts through to the edge where growth happens.

## What It Produces

The question locates exactly where vulnerability lives for this person in this moment. Because vulnerability is personal (what is vulnerable for one person is easy for another), this question adapts to whoever is asking it.

It produces one of:
- An admission the person has been avoiding ("I'm scared this isn't working")
- A feeling they have been suppressing ("I'm angry and I don't want to say it")
- A truth they know but have not spoken ("I don't want to be here")
- Sometimes silence, which is itself information

## What Joe Does With the Response

Joe does not push toward a specific answer. Whatever arises is the truth. If the person names it, Joe stays with it. If the person resists, the resistance itself is data about where the fear is.

## Tone

Warm, quiet. Not challenging. This is an invitation, not a demand.
