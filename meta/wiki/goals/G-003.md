# Download all source media files

## Outcome

Every item in the catalog (from [[G-002]]) has its media file downloaded locally. Audio files for podcast episodes, video or audio files for YouTube content. Stored in `transcripts/{YYYY-MM-DD}_{title}/` directory structure. Each directory has a `MEDIA.md` stub with title and date at minimum.

## Context

Depends on [[G-002]] catalog being substantially complete. Can start downloading as catalog entries are confirmed, does not need to wait for 100% catalog completion. Platform-specific download tools (yt-dlp for YouTube, podcast feed downloaders for RSS) will be needed. Some tools may have changed or broken since May 2025.
