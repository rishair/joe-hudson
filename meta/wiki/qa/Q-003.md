# QA: Warm-somatic redesign: visual fidelity desktop+mobile, no functional regression, selector still works

## Scope

The [[E-055]] redesign: the warm-somatic theme is applied, the [[Q-001]] mobile defects are fixed, and no prior functionality regressed (chat/streaming, resource modal, persistence, welcome, wiki, coach-variant selector).

## Test plan

Golden path, then mobile, then regression.

1. **Theme applied (desktop).** Load the deployed app. Confirm: non-default fonts render (Fraunces display + the chosen body face — NOT Inter/Roboto/Arial/system), warm-neutral background, deep ink text, single warm accent. No purple-on-white, no flat default shell. Screenshot the chat, welcome, and wiki views.
2. **Mobile fidelity + Q-001 fixes.** At 412px and 375px: no horizontal overflow (scroll width == viewport width), the sidebar adapts/collapses, the chat input is full-width and usable, the selector is reachable. Screenshot each.
3. **No functional regression.** Send a coach turn: streaming renders progressively ([[E-052]]); the "N sources" modal opens and shows resources ([[E-044]]); a reply persists across reload ([[E-041]]); the welcome flow shows on first visit ([[E-045]]); the wiki view renders and wikilinks navigate ([[E-042]]).
4. **Selector intact.** The coach-variant selector ([[E-054]]) still switches variants and the chosen variant drives the reply.
5. **Motion.** Page-load orchestration plays once and is not janky; no layout shift from the staggered reveal.

### Edge cases
- Tablet width (~768px): layout transitions cleanly, no broken intermediate state.
- Reduced-motion preference: animations respect `prefers-reduced-motion` if implemented (note if not).
- Bundle/asset weight: confirm fonts did not blow the Cloudflare bundle ceiling (deploy succeeded).

## Pre-conditions

- [[E-055]] deployed live to the Cloudflare URL (confirm the redeploy landed; URL currently `https://joe-hudson-coach.rishi-ishairzay.workers.dev`).
